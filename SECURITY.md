# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it immediately.

Do not open a public issue. Instead, please send an email to the project maintainers or use the GitHub security advisory feature if enabled. We will try to respond to the issue as soon as possible and take the appropriate steps to resolve it.
